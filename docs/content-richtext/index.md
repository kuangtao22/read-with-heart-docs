# 📚 正文富文本

> **让你的电子书正文更精美、更富有表现力！**
>
> 用心读书正文支持 HTML 标签、CSS 样式、图片、段评、批注等富文本能力。本栏目按"基础样式"和"自定义标签"两大类组织文档。

---

## 📖 文档结构

### 基础样式

[基础样式 →](basic.md) 涵盖 HTML 标签、CSS 样式表、完整示例、最佳实践与快速上手模板。

### 自定义标签

用心读书在标准 HTML 之外扩展了三个自定义标签，用于实现图片渲染、段评、批注等交互能力：

| 标签 | 用途 | 入口 |
|------|------|------|
| `<img>` | 图片插入（网络图、SVG、可点击） | [图片 →](image.md) |
| `<comment>` | 段落评论按钮 | [段评 →](comment.md) |
| `<note>` | 行间批注气泡（远程弹窗 / 本地批注） | [批注 →](note.md) |

---

## ✨ 支持的功能

✅ **HTML 标签**：`<b>`、`<i>`、`<u>`、`<span>`、`<h1>`-`<h6>`、`<p>`、`<br>` 等  
✅ **CSS 样式**：颜色、字体、大小、对齐、行高、间距等  
✅ **图片**：自动缩放、比例保持、占位符显示  
✅ **段评**：段落评论按钮、点击交互  
✅ **批注**：行间批注气泡，支持远程弹窗与本地批注  

---
